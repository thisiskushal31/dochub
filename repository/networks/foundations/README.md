# Foundations

- Models: OSI vs TCP/IP, encapsulation/decapsulation.
- Addressing & subnetting: IPv4/IPv6, CIDR, VLSM basics.
- Switching: MAC learning, VLANs, trunking, spanning tree fundamentals.
- Link layer: duplex, MTU, Ethernet framing, ARP/ND.
- Reliability: link aggregation (LACP), redundancy patterns.
